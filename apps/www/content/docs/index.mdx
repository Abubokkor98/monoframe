# Monoframe

**One command. A production-ready monorepo. Zero config headaches.**

Monoframe is a CLI tool that generates fully configured Turborepo monorepo projects with Next.js, shadcn/ui, animation libraries, and CI/CD — so you can skip the setup and start building.

## What You Get

Every project scaffolded by Monoframe includes:

- **Turborepo** monorepo with optimized task pipelines
- **Next.js** frontend apps with TypeScript and Tailwind CSS v4
- **Express + TypeScript** backend apps
- **Shared packages** for UI components, ESLint, TypeScript, and Tailwind configs
- **shadcn/ui** in a monorepo-optimized shared package with custom presets
- **Animation libraries** — Framer Motion, Lenis, GSAP — pre-wired with providers
- **Code quality** — ESLint + Prettier or Biome, with Husky + lint-staged
- **CI/CD** — GitHub Actions pipelines and Playwright e2e testing

## Quick Start

```bash
npx monoframe init
```

Answer a few questions about your project — app names, frameworks, features — and Monoframe handles the rest.
