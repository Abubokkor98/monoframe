## When to Use Monoframe

**Use Monoframe when you need:**

- A monorepo with multiple apps sharing code
- Production-ready tooling from day one
- shadcn/ui components shared across apps
- Animation libraries pre-configured
- CI/CD pipeline without manual setup

**Use something else when you need:**

- A single-app project (use `create-next-app`)
- tRPC / database / auth integration (use `create-t3-app`)
- Nx-specific features (Nx support coming in v1.1)
- Non-Next.js frameworks (multi-framework support coming in v2.0)
